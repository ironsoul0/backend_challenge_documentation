Marketplace

This is a documentation for API which is a part of Shopify Backend Challenge. Base URL for this API is https://shopify-challenge-backend.herokuapp.com

Get all products

Getting all products available in marketplace. Also you can pass an argument to only return products with available inventory.

Property Description
only_available (optional) should be equal to '1' if you want to only return products with available inventory.
GET
/products/all_products

Example request body

{
  "only_available": "1"
}

Example response

[
    {
        "_id": "5c3d627f2778a902af7c5696",
        "title": "water",
        "price": 30,
        "inventory_count": 4,
        "__v": 0
    },
    {
        "_id": "5c3d635a145ce802bd2dd095",

Single product

Returns information about one product based on its title.

Property Description
title (required) title of the product.
GET
/products/single_product

Example request body

{
  "title": "duck"
}

Example response (product with such title exists)

{
    "_id": "5c3d95655fea3302750818a3",
    "title": "duck",
    "price": 11,
    "inventory_count": 2,
    "__v": 0
}

Example response (product with such title does not exist)

{
    "message": "No product with such title"
}

Purchase a product

Decreasing the inventory count of the product by 1, if possible.

Property Description
title (required) title of the product.
PUT
/products/purchase_product

Example request body

{
  "title": "milk"
}

Example response (product with such title exists and inventory count is greater than 0)

{
    "message": "Purchase completed",
    "new_inventory_count": 8
}

Example response (product with such title exists and inventory count equals to 0)

{
    "message": "Inventory count of the product is zero"
}

Example response (product with such title does not exist)

{
    "message": "No product with such title"
}
